## 🪟 Bespoke Glass + Deep Glass — the material system that responds to you

This isn't a CSS effect. It's a material. Two languages — both real-physics, both calibrated, both gated by an audit before they ship. Most "glass" you've seen on the web is a one-line backdrop-filter trick. Bespoke Glass is built like an actual product material, and Deep Glass is what you reach for when the moment deserves cinema.

### 🧊 Bespoke Glass — the everyday surface

A real, layered backdrop-filter at 20px with an edge-bleed restraint, sub-pixel chamfer on the edges so they don't look hard-cut against the surface behind them, and a subtle gel-grain texture that catches light the way actual frosted acrylic does — not the smudgy "frosted" look from generic blur. Color isn't a flat tint; it samples from the surface beneath, so Glass over sage looks sage, over crystal looks crystal. Atmospheric density responds to layering depth — a Glass card sitting on a Glass panel reads as nested, not stacked-flat.

Use it for: chrome, chips, modals, cards, navigation, buttons — anywhere you want premium depth without atmospheric drama.

### 🌌 Deep Glass — the cinematic tier

Everything Bespoke Glass has, plus the things that make people stop scrolling and reach out to touch the screen:

- **Time-of-day atmosphere** — the surface tint shifts with phase-of-day. Dawn is warmer-amber, mid-day is bright-neutral, dusk drifts toward rose, night is deep-indigo, midnight is near-black with cool accents. Not a theme switcher. A real lighting model.
- **Weather through the glass** — clear / overcast / rain / snow / storm, animated subtly so the surface feels alive without distracting. The atmosphere actually moves behind the chrome. Sync to your real local weather if you want it.
- **Edge-bleed on hero surfaces** — light spills through the glass where it meets the world behind it. This is the Bespoke crystal signature.
- **Family-accent chrome** — vertical accent bars that pull from your brand palette across the four rooms of The Bespoke Studio (The Goldroom, The Silver Floor, The Violet Studio, The Pearl Library).
- **Touch-reactive gel ripple** (iOS / mobile web) — when you tap a Deep Glass button, a real-physics ripple emanates from your touch point and refracts through the glass material. Built with SwiftUI + Metal shader on native; CSS+JS gel-physics on web. Fades over ~400ms. Feels alive in your hand.
- **Mouse-following gel glow** (desktop web) — the glass softly highlights where your cursor approaches, like the surface is responding to you. Not a hover state. A continuous proximity glow.
- **Iridescent diagonal light streaks** — a 45-second ambient drift across the surface so it never feels static, but never distracts either.
- **Pearl Library bowl interior** — the rare-tier characters have a translucent pearl bowl that holds light differently depending on viewing angle. Real 3D depth, not a sticker.

Use it for: hero surfaces, landing pages, splash screens, pitch decks, brand-flagship work — anywhere the visual register itself is the message.

### 🍎 How it's different from Apple's Liquid Glass

Apple's Liquid Glass (iOS 18+) is a system effect — a backdrop blur with slight refraction, applied uniformly by the OS. It looks nice. Bespoke Glass and Deep Glass go several layers deeper:

| | Apple Liquid Glass | Bespoke Glass / Deep Glass |
|---|---|---|
| Depth | Flat blur | Real atmospheric depth + time/weather sync |
| Tint | System-tinted | Brand-aware tinting per kit |
| Cost control | Full-blur everywhere | Render-cost contract — max one full-blur surface visible at once, nested falls back |
| Audit | None | Pre-Render Audit gates every surface before ship |
| Customization | None | 4 family palettes, per-brand tokens, per-surface overrides |
| Interaction | Visual only | Touch-reactive gel ripple (mobile) + proximity glow (desktop) |
| Anti-pattern aware | No | Tell Bespoke "no glass on invoices" once → it sticks per v98 |
| When it shows up | Always | Per v100 aesthetic-fit — only where it serves the work |
| Per-brand identity | One look | Four family aesthetics + custom kit application |

Apple's is a system effect. Bespoke is a design language with audit + taste + brand-memory backing it.

### 🪞 The discipline behind it

Every Glass surface passes a **Pre-Render Audit** (glass-021 v2) — seven binary checks before ship: mobile parity, contains/portal safety, type-fit, reduced-motion safeguard, reduced-transparency fallback, weather animation budget, edge-bleed restraint. No surface ships if it fails any. That's why Bespoke Glass feels different from generic blur — it's gated by an audit, not just a CSS class.

Plus the render-cost contract (v89): max one full-blur surface visible at once. Nested glass uses a cheaper fallback. Atmosphere is GPU-budgeted per layer. Your phone doesn't melt.

### How you can use it

- **Greenfield builds:** import the CSS token system, drop in the templates. Done.
- **Retrofitting an existing site:** Bespoke does a surface walk, identifies which elements should become Glass vs Deep Glass vs flat, and produces a surgical patch you paste in.
- **Other tools:** the token system works in React, SwiftUI, vanilla CSS, Figma. Bespoke generates the right form for the platform you're on.

### 🎨 When Glass does NOT show up

Bespoke Glass and Deep Glass are LANGUAGES — not the default applied to every output. An invoice gets clean-flat readability. A daily email gets warm-minimal. A pitch deck might get Deep Glass. The cast picks the right register per surface.

Glass and Deep Glass become MORE special because they're used where they matter — not because they're wallpapered onto everything.

Want Glass everywhere? Tell Bespoke once — it sticks. Want never? Same. Bespoke respects your taste, not its own preferences.

Ask any Bespoke session: *"explain Glass and Deep Glass"* — Bespoke walks you through what each is, when each helps, and when neither fits.
